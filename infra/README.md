# Вспомогательная инфраструктура

Эта папка содержит вспомогательные артефакты для security-автоматизации и выполнения CI-пайплайна.

- `zap/` - опциональные правила ZAP, контекст и конфигурация baseline-сканирования
- `ci/` - вспомогательные скрипты для локальной эмуляции пайплайна (при необходимости можно расширять)
